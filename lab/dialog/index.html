方法

事件 描述
show([element]) 显示对话框

参数类型为HTMLElement:可吸附于元素上，同时呈气泡样式
showModal([element]) 显示一个模态对话框。同show
close() 关闭（隐藏）对话框。

删除请使用remove()方法
remove() 销毁对话框
content(html) 写入对话框内容
title(text) 写入对话框标题
width(value) 设置对话框宽度
height(value) 设置对话框高度
reset() 手动刷新对话框位置
button(args) 自定义按钮
focus() 聚焦对话框（置顶）
blur() 让对话框失去焦点
ZSdialog.get(id) 根据id获取打开的对话框的实例
dialog.getCurrent() 获取当前（置顶）对话框实例

参数

参数 类型 描述
title String 标题内容
content String 设置消息内容
ok Function 确定按钮

回调函数this指向ZSdialog对象，执行完毕默认关闭对话框，若返回 false 则阻止关闭。
okValue String 确定按钮文本。默认值：'ok'
cancel Function, Boolean 取消按钮

等同于标题栏关闭按钮。值为 false 则不显示关闭按钮。
cancelValue String 取消按钮文本。默认值：'cancel'
cancelDisplay Boolean 是否显示取消按钮。默认值：true
button Array 自定义按钮组
参数 类型 描述
value String 按钮显示文本
callback Function 回调函数。this指向ZSdialog对象。执行完毕默认关闭与销毁对话框。若返回 false 则阻止关闭与销毁。
autofocus Boolean 是否自动聚焦。默认：false
disabled Boolean 是否禁用。默认：false
width String, Number 设置对话框宽度
height String, Number 设置对话框高度
skin String 设置对话框额外的className参数
padding String, Number 设置消息内容与消息容器的填充边距
fixed Boolean 开户固定位置。默认值：false。
align String 设置对话框与其它元素的对齐方式。默认值：'bottom left'

支持格式：

'top left', 'top', 'top right', 'right top', 'right', 'right bottom', 'bottom right', 'bottom', 'bottom left', 'left bottom', 'left', 'left top'
autofocus Boolean 是否支持自动聚焦。默认值：true
quickClose Boolean 是否点击空白处快速关闭。默认值：false
zIndex Number 重置全局zIndex初始值，用来改变对话框叠加高度。默认值：1024

注意：会影响全局配置。后续出现的对话框高度叠加以此为准。
onshow Function 对话框打开事件。回调函数 this 指向ZSdialog
onclose Function 对话框关闭后执行事件
onbeforeremove Function 对话框销毁前事件
onremove Function 对话框销毁事件
onfocus Function 对话框获得焦点事件
onblur Function 对话框失去焦点事件
onreset Function 对话框位置重置事件
id String 设置对话框唯一标示。

可防止重复ID对话框弹出。支持使用ZSdialog.get(id)获取某个对话框
backdropBackground String 遮罩层颜色值。默认值：'#000'
backdropOpacity Number 遮罩层半透明度。默认值：0.55
className String 弹框默认样式。默认值：'ZSdialog'

属性

属性名 类型 描述
open Boolean 判断对话框是否被打开

参考

普通对话框

var d = ZSdialog({
    title: 'welcome',
    content: 'welcome use ZSdialog'
});
d.show();

模态对话框

var d = ZSdialog({
    title: 'welcome',
    content: 'welcome use ZSdialog'
});
d.showModal();

气泡浮层

var d = ZSdialog({
    content: 'welcome use ZSdialog',
    quickClose: true
});
d.show(document.getElementById('bubble'));

确定与取消按钮

var d = ZSdialog({
	title: '提示',
	content: '带有按钮的弹出框',
	okValue: 'ok',
	ok: function(){
		this.title('提交中...');
		return false;
	},
	cancelValue: 'cancel',
	cancel: true
});
d.showModal();
